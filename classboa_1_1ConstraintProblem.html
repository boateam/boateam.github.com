map< Buffer, vector< string > > boa::ConstraintProblem::SolveAndBlame (  )  const

Solve the constraint problem and generate a minimal set of constraints which cause each overrun

Return a map - the keys are possibly overrun buffers, the corresponding value is a small set of constraints which cause the overrun. The set is minimal in the sense that no subset of these constraint will cause the specific buffer overrun, there might be other (smaller) set which will also cause the overrun.
