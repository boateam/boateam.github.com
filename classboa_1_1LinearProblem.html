void boa::LinearProblem::RemoveInfeasable (  ) 

Remove a minimal set of constraints that makes the lp inFeasable.

Each of the variables in the removed rows will become "unbound" (e.g. - ...!max >= MAX_INT or ...!min <= MIN_INT).

The removed set is *A* minimal in the sense that these lines alone form an inFeasable problem, and each subset of them does not. It is not nessecerily *THE* minimal set in the sense that there is no such set which is smaller in size.
