Detailed Description

Model a single constraint.

A constraint is a simple linear inequality of the form BIG >= SMALL Use addBig in order to add an expression, literal or a constant to the BIG side of the inequality, or the equivalent functions for the SMALL side.

For internal use, the constraint is stored in the form of -

C >= aX + bY ...

Where C is a constant (referred as "left"), a small letter (a, b...) is an integer value ("num") and a capital letter (X, Y...) is a string name of variable.

enum boa::Constraint::Type

STRUCTURAL - Created as part of boa's internal operation and not because of user code.

ALIASING - A constraint for buffer aliasing, less important in boa's output

NORMAL - Any other constraint, these constraints has a high probability of being the cause for a buffer overrun.
